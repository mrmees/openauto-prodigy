**Timer:**
- `interval: 60000` (1-minute, date doesn't need seconds)
- `running: dateWidget.isCurrentPage`
- `repeat: true`, `triggeredOnStart: true`
- On trigger, capture `new Date()` and extract: `dayOfMonth` (int), `dayOfWeekShort` (3-letter, e.g. "Mon"), `dayOfWeekLong` (e.g. "Monday"), `monthShort` (3-letter, e.g. "Aug"), `monthLong` (e.g. "August")

**Ordinal suffix function:**
```javascript
function ordinalSuffix(n) {
    var s = ["th", "st", "nd", "rd"];
    var v = n % 100;
    return n + (s[(v - 20) % 10] || s[v] || s[0]);
}
```

**Column-driven breakpoints — single Item layout, content switches on colSpan:**

- **colSpan 1 (1x1):** Hero layout. Show `ordinalSuffix(dayOfMonth)` centered, bold (`Font.Bold`), large (`UiMetrics.fontHeading * 3.0`), `fontSizeMode: Text.Fit`, `minimumPixelSize: UiMetrics.fontHeading`. Color: `ThemeService.onSurface`. This IS the widget at 1x1 — "desk calendar tear-off" feel.

- **colSpan 2 (2x1):** Row layout. Show `dayOfWeekShort + " " + ordinalSuffix(dayOfMonth)`. E.g. "Mon 21st". Single NormalText, `font.pixelSize: UiMetrics.fontHeading * 2.0`, `Font.Bold`, `fontSizeMode: Text.Fit`, centered.

- **colSpan 3 (3x1):** Row layout. Show `dayOfWeekLong + ", " + monthShort + " " + ordinalSuffix(dayOfMonth)` for US order, or `dayOfWeekLong + ", " + ordinalSuffix(dayOfMonth) + " " + monthShort` for intl order. E.g. "Tuesday, Aug 10th" or "Tuesday, 10th Aug". Single NormalText, `font.pixelSize: UiMetrics.fontHeading * 1.5`, day-of-week in `Font.Normal`, rest `Font.Bold` — actually since it's one Text element, use `Font.Bold` overall with the day number being the naturally prominent part.

- **colSpan 4+ (4x1+):** Full detail. Same as 3x1 but with `monthLong` instead of `monthShort`. E.g. "Monday, December 31st" (US) or "Monday, 31st December" (intl).

**Height >= 2 behavior:** When `rowSpan >= 2`, increase font sizes by 1.5x multiplier (scale up the column-gated content). Center vertically with `anchors.centerIn: parent`.

**Layout implementation:** Use a single `Item` anchored to fill parent with padding (`UiMetrics.spacing`). Inside, a `NormalText` element whose `text` property is a JS expression switching on colSpan and dateOrder. Use `fontSizeMode: Text.Fit` with appropriate `minimumPixelSize` to handle all spans gracefully.
