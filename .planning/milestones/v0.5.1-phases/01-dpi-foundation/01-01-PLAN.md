  <name>Task 1: Extend DisplayInfo with screen size and DPI properties</name>
  <files>src/ui/DisplayInfo.hpp, src/ui/DisplayInfo.cpp, tests/test_display_info.cpp</files>
  <behavior>
    - Default screenSizeInches is 7.0
    - setScreenSizeInches(10.1) updates screenSizeInches to 10.1
    - setScreenSizeInches(0) and setScreenSizeInches(-1) are ignored (stays at previous value)
    - setScreenSizeInches with same value does not emit signal
    - computedDpi for 1024x600 window at 7.0" = round(hypot(1024,600) / 7.0) = round(1186.7/7.0) = 170
    - computedDpi for 1280x720 window at 10.1" = round(hypot(1280,720) / 10.1) = round(1468.6/10.1) = 145
    - Changing window size emits screenSizeChanged is NOT expected (windowSizeChanged only), but computedDpi value updates
    - screenSizeChanged signal emits when setScreenSizeInches called with new valid value
  </behavior>
  <action>
    Add to DisplayInfo.hpp:
    - Q_PROPERTY(qreal screenSizeInches READ screenSizeInches NOTIFY screenSizeChanged)
    - Q_PROPERTY(int computedDpi READ computedDpi NOTIFY windowSizeChanged)
    - Note: computedDpi NOTIFY uses windowSizeChanged because it depends on window dimensions AND screen size. Also emit windowSizeChanged from setScreenSizeInches to trigger DPI recalc in QML.
    - Add qreal screenSizeInches() const, int computedDpi() const, void setScreenSizeInches(qreal inches)
    - Add signal: void screenSizeChanged()
    - Add member: qreal screenSizeInches_ = 7.0

    In DisplayInfo.cpp:
    - #include <cmath> for std::hypot and std::round
    - screenSizeInches(): return screenSizeInches_
    - computedDpi(): return static_cast<int>(std::round(std::hypot(windowWidth_, windowHeight_) / screenSizeInches_))
    - setScreenSizeInches(qreal inches): guard <=0, guard qFuzzyCompare, set, emit screenSizeChanged(), emit windowSizeChanged()

    In tests/test_display_info.cpp:
    - Write tests FIRST per TDD. Add tests for: default screenSizeInches, setScreenSizeInches updates, zero/negative ignored, same-value no signal, computedDpi at 1024x600/7.0", computedDpi at 1280x720/10.1".
