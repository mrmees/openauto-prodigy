    Three changes to install.sh:

    **1. Add SCREEN_SIZE default (near line 76 with other defaults):**


    **2. Add probe_screen_size() function** (place near other utility functions, before setup_hardware):

    Parse raw EDID from `/sys/class/drm/*/edid`:
    - Find first connected display (check `status` file == "connected", EDID file exists and is non-empty)
    - Try detailed timing descriptor first (bytes 66-68 for mm precision):
      - Check pixel clock at bytes 54-55 is non-zero (confirms timing descriptor, not monitor descriptor)
      - Extract 12-bit width_mm and height_mm from bytes 66-68
    - Fall back to basic display params (bytes 21-22 for cm precision)
    - Convert to diagonal inches: `awk "BEGIN {printf \"%.1f\", sqrt(w*w + h*h) / 25.4}"`
    - Return empty string if no EDID available or dimensions are zero
    - Use `od -An -tu1` for byte reading (no `bc`, no extra packages)

    **IMPORTANT bash gotchas:**
    - Use `if/then/else` not `[[ ]] && action` (set -e kills script on false condition)
    - No `grep -oP` (Perl regex unavailable on minimal Trixie)
    - No `bc` -- use `awk` for float math
    - `od` output has leading spaces -- use `tr -d ' '` to clean

    **3. Add screen size detection to setup_hardware():**

    Insert AFTER the touch device section (after line 885 `fi`), BEFORE the WiFi AP section.
    Add `clear_body` and `tput cup "$HEADER_ROWS" 0` first (same pattern as WiFi section).

    Flow:
    ```
    clear_body
    tput cup "$HEADER_ROWS" 0
    info "Detecting display..."
    DETECTED_SIZE=$(probe_screen_size)

    if [[ -n "$DETECTED_SIZE" ]]; then
        # EDID detected -- compute mm dimensions for display
        ok "Detected screen: ${DETECTED_SIZE}\""
        echo ""
        read -p "Press Enter to accept, or type a different size in inches: " USER_SIZE
        if [[ -n "$USER_SIZE" ]]; then
            # Validate numeric input
            if [[ "$USER_SIZE" =~ ^[0-9]+\.?[0-9]*$ ]] && awk "BEGIN {exit !($USER_SIZE > 0)}"; then
                SCREEN_SIZE="$USER_SIZE"
            else
                warn "Invalid input, using detected size"
                SCREEN_SIZE="$DETECTED_SIZE"
            fi
        else
            SCREEN_SIZE="$DETECTED_SIZE"
        fi
    else
        info "Could not detect screen size from EDID"
        echo ""
        read -p "Enter screen diagonal in inches [7.0]: " USER_SIZE
        if [[ -n "$USER_SIZE" ]]; then
            if [[ "$USER_SIZE" =~ ^[0-9]+\.?[0-9]*$ ]] && awk "BEGIN {exit !($USER_SIZE > 0)}"; then
                SCREEN_SIZE="$USER_SIZE"
            else
                warn "Invalid input, using default 7.0\""
                SCREEN_SIZE="7.0"
            fi
        fi
        # else SCREEN_SIZE stays at default "7.0"
    fi
    ok "Screen size: ${SCREEN_SIZE}\""
    ```

    **4. Update generate_config():**

    In the heredoc config output (around line 1281), add `screen_size` to the display section:
    ```yaml
    display:
      screen_size: ${SCREEN_SIZE}
      width: 1024
      height: 600
      brightness: 80
    ```

    Note: `width` and `height` in config may be vestigial (app reads window dimensions at runtime),
    but don't remove them in this plan -- that's cleanup for later.
