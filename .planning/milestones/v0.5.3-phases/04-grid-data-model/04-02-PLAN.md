  <action>
    **Create src/ui/WidgetGridModel.hpp:**
    Follow the API from RESEARCH.md Pattern 1. Key design:
    - QAbstractListModel with roles: WidgetIdRole, InstanceIdRole, ColumnRole, RowRole, ColSpanRole, RowSpanRole, OpacityRole, QmlComponentRole, VisibleRole
    - Q_PROPERTY gridColumns/gridRows with gridDimensionsChanged signal
    - Q_INVOKABLE: placeWidget, moveWidget, resizeWidget, removeWidget, setWidgetOpacity, canPlace
    - Internal: OccupancyGrid (Pattern 2 from research — QVector<QString> cells, instanceId per cell)
    - `placements()` returns QList<GridPlacement> for serialization
    - `setPlacements(QList<GridPlacement>, WidgetRegistry*)` for loading from config — needs registry to resolve QML component URLs
    - Private: `int nextInstanceId_ = 0` for monotonic instance ID generation
    - `setGridDimensions(int cols, int rows)` triggers clamping algorithm (Pattern 4)

    **Create src/ui/WidgetGridModel.cpp:**
    - `placeWidget`: validate bounds + canPlace, generate instanceId (widgetId + "-" + nextInstanceId_++), beginInsertRows/endInsertRows, mark occupancy, emit placementsChanged
    - `moveWidget`: find placement by instanceId, validate new position with canPlace(exclude=self), clearRegion old, update coords, markOccupied new, emit dataChanged + placementsChanged
    - `resizeWidget`: validate against widget's min/max (lookup from registry), validate bounds, validate canPlace for new region (exclude=self), update spans, rebuild occupancy, emit dataChanged + placementsChanged
    - `removeWidget`: find by instanceId, beginRemoveRows/endRemoveRows, rebuild occupancy, emit placementsChanged
    - `canPlace`: check bounds, iterate region cells in occupancy grid, skip cells owned by excludeInstanceId
    - `setGridDimensions`: implement clamping algorithm from research Pattern 4. Sort by insertion order (list order). Clear occupancy. For each placement: clamp spans to min(span, newDim), clamp position to max(0, newDim - span), check overlap — if free mark occupied+visible, if occupied mark visible=false. Emit gridDimensionsChanged + placementsChanged.
    - `setWidgetOpacity`: find placement, update opacity, emit dataChanged for OpacityRole
    - Store `WidgetRegistry*` pointer for descriptor lookups (size constraints in resize)

    **src/CMakeLists.txt:** Add WidgetGridModel.cpp to openauto-core sources
