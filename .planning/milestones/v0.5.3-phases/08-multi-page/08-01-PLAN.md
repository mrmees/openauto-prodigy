<objective>
Add page awareness to WidgetGridModel and YAML persistence. Every grid operation (place, move, resize, collision check) becomes page-scoped. Page count is tracked and persisted. This is the data layer that Plan 02's QML SwipeView will consume.

Purpose: Without page-scoped operations, all widgets exist on a single flat grid. Multi-page navigation is meaningless if the model can't distinguish which page a widget belongs to.
Output: Page-aware WidgetGridModel, schema v3 YAML persistence, unit tests proving page isolation.
</objective>
