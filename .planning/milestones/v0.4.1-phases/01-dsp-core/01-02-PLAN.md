  <action>
    **Header (`src/core/audio/EqualizerEngine.hpp`):**

    In `namespace oap`, create `class EqualizerEngine` (NOT a QObject — pure DSP, no Qt signals needed):

    **Public API:**
    - `EqualizerEngine(float sampleRate = 48000.0f, int channels = 2)` — initialize 10 bands at 0dB, bypass off
    - `void setGain(int band, float dB)` — called from non-RT thread. Clamp to ±12dB. Compute new coefficients via `calcPeakingEQ`, atomic-swap to RT thread. If interpolation is in progress, capture current interpolated coefficients as new "old" before starting new ramp.
    - `void setAllGains(const std::array<float, 10>& gainsDb)` — set all 10 bands at once (preset switch). Single atomic swap for all bands. Same interpolation ramp.
    - `float getGain(int band) const` — return current gain for band
    - `void setBypassed(bool bypassed)` — atomic flag. Triggers wet/dry crossfade ramp.
    - `bool isBypassed() const`
    - `void process(int16_t* data, int frameCount)` — RT-safe. Process stereo interleaved int16_t frames in-place. Pipeline per frame: int16→float, 10 biquad stages, soft limiter, float→int16 with clamp. Handles interpolation and bypass crossfade internally.
    - `static constexpr int kInterpolationSamples = 2304` — ~48ms at 48kHz

    **Internal design (Claude's discretion per CONTEXT.md):**

    Use atomic double-buffer for coefficient swap:
    - `struct EngineCoeffs` — array of 10 `BiquadCoeffs` + bypass flag
    - Two `EngineCoeffs` buffers, `std::atomic<EngineCoeffs*>` for RT thread to read
    - Non-RT thread writes to inactive buffer, swaps pointer
    - RT thread detects swap at start of each `process()` call, begins interpolation

    Per-sample coefficient interpolation:
    - Store `oldCoeffs_[10]` and `newCoeffs_[10]` with `interpSamplesRemaining_` counter
    - Each sample: compute `t = 1.0 - (remaining / total)`, lerp each coefficient
    - When new coefficients arrive mid-ramp: snapshot current interpolated values as new `oldCoeffs_`

    Bypass crossfade:
    - `bypassRampRemaining_` counter, `bypassTarget_` (0.0=EQ active, 1.0=bypass)
    - Process BOTH wet (EQ'd) and dry paths during ramp, crossfade
    - When fully bypassed: skip all processing (save CPU)
    - When fully active: skip crossfade multiply

    Per-channel state:
    - `BiquadState states_[2][10]` — 2 channels × 10 bands
    - `SoftLimiter limiters_[2]` — one per channel

    Gain tracking:
    - `float gains_[10]` — current gain per band (for getGain())

    **Implementation (`src/core/audio/EqualizerEngine.cpp`):**
    - Implement all methods
    - `process()` must be fully RT-safe: no allocations, no mutexes, no logging
    - Use `std::memory_order_acquire` for RT-thread loads, `std::memory_order_release` for non-RT stores
    - Clamp int16_t output: `std::clamp(static_cast<int32_t>(sample * 32768.0f), -32768, 32767)`

    **CMake updates:**
    - Add `src/core/audio/EqualizerEngine.cpp` to `openauto-core` library sources in `src/CMakeLists.txt`
    - Add test to `tests/CMakeLists.txt`:
      ```cmake

      ```

    **Test (`tests/test_equalizer_engine.cpp`):**

    Qt Test class `TestEqualizerEngine` with tests for each behavior. Test approach:
    - Generate int16_t stereo sine waves at specific frequencies
    - Process through engine with various gain settings
    - Measure output RMS and compare to expected boost/cut
    - For interpolation smoothness: process frame-by-frame during transition, check max sample-to-sample delta
    - For bypass: compare output to input (should be within float epsilon of original)
    - For clipping: boost all bands +12dB on full-scale input, verify no int16_t overflow

    **Per user decisions (all from CONTEXT.md — NON-NEGOTIABLE):**
    - ±12dB range, 1dB steps (25 positions per band)
    - ~50ms crossfade → 2304 samples at 48kHz
    - Same transition speed for preset switches and single-band changes
    - Direct crossfade only — no volume dip
    - Smooth fade on bypass toggle (same 2304 sample ramp)
    - Global bypass only
    - Changes stored during bypass, applied on un-bypass
    - Default all bands 0dB (flat)
