### Default theme
- Prodigy becomes the new `default` theme (id: default, name: Prodigy or similar)

### Wallpapers
- Include wallpaper JPEGs from each Pi theme in the bundled theme directory
- Themes should look complete out of the box without companion app

### Color values
- Use companion-app-generated M3 palettes as-is — no manual color tweaking
- The companion app generates proper M3 color math; trust the output

### Claude's Discretion
