    2. **Control Categories:** action, selection, gesture, passive (per locked decision).
    3. **For each control, document a table:**

    | State | Container | Content | Border | Notes |
    |-------|-----------|---------|--------|-------|

    Controls to document:
    - **Action controls:** ElevatedButton, FilledButton, OutlinedButton, SettingsRow (tappable)
    - **Selection controls:** SettingsToggle (Switch), Tile, theme selector, ComboBox/picker
    - **Gesture controls:** NavbarControl (rest, hold-progress, active states)
    - **Passive content:** NormalText, SpecialText, SettingsRow (display-only), dividers, page dots

    Specific token assignments per control (matching what's implemented):
    - **NavbarControl rest:** surfaceContainer bg (via barBg), onSurface icon (via barFg)
    - **NavbarControl hold-progress:** primaryContainer fill overlay at 0.3 opacity, onPrimaryContainer icon
    - **NavbarControl (AA active):** #000000 bg, #FFFFFF icon (hardcoded, AA owns display)
    - **ElevatedButton rest:** surfaceContainerLow bg, barFg text
    - **ElevatedButton pressed:** primaryContainer bg, onPrimaryContainer text
    - **Tile rest:** primary bg, onPrimary content
    - **Tile pressed:** primaryContainer bg, onPrimaryContainer content
    - **SettingsToggle:** neutral row, Switch control follows Qt Quick Controls default theme (future: custom M3 Switch style)
    - **Settings rows:** surfaceContainer bg (neutral), onSurface text -- NO primaryContainer fill per "don't Skittle the settings" decision
    - **Popup/dialog surfaces:** surfaceTintHigh (opaque) or surfaceTintHighest with 0.87 alpha
    - **Widget cards:** surfaceContainer glass (from WidgetHost), accent via text/icons only (primary for values/icons, onSurface/onSurfaceVariant for labels)
    - **AAStatusWidget connected:** success (semantic green), NOT primary
    - **Page dots:** onSurfaceVariant at 0.4 opacity

    6. **Widget accent rules recap:** one accent family per widget, primary for default, primaryContainer reserved for state changes.
