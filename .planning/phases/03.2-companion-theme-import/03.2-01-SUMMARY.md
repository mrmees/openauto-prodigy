## Decisions Made
- **scrim from YAML:** Changed from hardcoded `QColor(0,0,0,180)` to `activeColor("scrim")` with opaque black fallback. QML sites that need transparency should apply alpha themselves.
- **error vs red:** Both `error`/`on-error` (M3) and `red`/`on-red` (AA custom) coexist. Bundled themes set both. Companion themes use M3 names.
- **Theme protection:** `deleteTheme` uses `searchPaths_.first()` (user themes dir) as the writable boundary. Themes outside it are protected.
- **allThemeKeys union:** `persistConnectedDeviceTheme` now uses `knownM3Keys | knownAATokenKeys` so all roles are preserved.
