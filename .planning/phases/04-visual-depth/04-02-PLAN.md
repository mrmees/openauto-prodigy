  <action>
Create three M3-inspired button components in `qml/controls/`. All three share the same core structure (MultiEffect shadow + state layer + press animation) but differ in visual style.

**Common properties for all three:**
- `property string text: ""`
- `property string iconCode: ""` (Material Symbols codepoint)
- `property color buttonColor` (default varies by type)
- `property color pressedColor: ThemeService.primaryContainer`
- `property color textColor` (default varies by type)
- `property color pressedTextColor: ThemeService.onPrimaryContainer`
- `property real buttonScale: 0.97` (press scale, override to 0.95 for larger buttons)
- `property bool buttonEnabled: true`
- `property real buttonRadius: UiMetrics.radiusSmall`
- `property int elevation: 2` (M3 level: 0=flat, 1=subtle, 2=resting, 3=raised)
- `signal clicked()`
- `signal pressAndHold()`

**Shadow behavior (all three):**
Use `MultiEffect` with `import QtQuick.Effects`:
- **Resting (elevation 2 -- per user decision "Level 2 medium intensity"):** `shadowBlur: 0.50`, `shadowVerticalOffset: 4`, `shadowOpacity: 0.30`
- **Pressed:** `shadowBlur: 0.25`, `shadowVerticalOffset: 2`, `shadowOpacity: 0.15`
- Elevation 3 (for popups/dialogs when overridden): `shadowBlur: 0.70`, `shadowVerticalOffset: 6`, `shadowOpacity: 0.35`
- Elevation 1: `shadowBlur: 0.35`, `shadowVerticalOffset: 2`, `shadowOpacity: 0.25`
- Elevation 0: shadow disabled
- `shadowColor: ThemeService.shadow`
- `shadowHorizontalOffset: 0`
- `shadowScale: 1.0`
- Animate shadow properties with `Behavior { NumberAnimation { duration: UiMetrics.animDurationFast } }`

**Scale behavior (all three):**
- `scale: mouseArea.pressed && buttonEnabled ? buttonScale : 1.0`
- `Behavior on scale { NumberAnimation { duration: UiMetrics.animDurationFast; easing.type: Easing.OutCubic } }`

**State layer (all three):**
- Rectangle overlay anchored to bg, same radius
- `color: ThemeService.onSurface`
- `opacity: mouseArea.pressed ? 0.10 : 0.0`
- `Behavior on opacity { NumberAnimation { duration: UiMetrics.animDurationFast } }`

**Disabled state (all three):**
- `bg.opacity: buttonEnabled ? 1.0 : 0.5`
- MouseArea ignores clicks when `!buttonEnabled`

**Content layout (all three):**
- Row { spacing: UiMetrics.spacing/2; anchors.centerIn: parent }
- If iconCode set: MaterialIcon with the codepoint, sized to text height
- If text set: NormalText with the text
- Text color swaps between textColor and pressedTextColor on press

**ElevatedButton specifics:**
- Default `buttonColor: ThemeService.surfaceContainerLow`
- Default `textColor: ThemeService.primary`
- Default `elevation: 2`
- This is the general-purpose "raised card" button

**FilledButton specifics:**
- Default `buttonColor: ThemeService.primary`
- Default `textColor: ThemeService.onPrimary`
- Default `pressedColor: ThemeService.primaryContainer` (lighter on press, not darker)
- Default `pressedTextColor: ThemeService.onPrimaryContainer`
- This is for primary action buttons (save, confirm, etc.)

**OutlinedButton specifics:**
- Default `buttonColor: "transparent"`
- Default `textColor: ThemeService.primary`
- Border: `border.width: 1; border.color: ThemeService.outline`
- Default `elevation: 0` (no shadow at rest, shadow appears on press at level 1 values)
- On press: elevation visually transitions to 1 (shadow appears)
- This is for secondary/cancel actions

**IMPORTANT:** The MultiEffect `source` must be the background Rectangle, and the MultiEffect must be a sibling (not parent) of the source. The source Rectangle should have `layer.enabled: true` to work properly with MultiEffect if needed. Test that autoPaddingEnabled (default true) provides enough space for shadow rendering.
