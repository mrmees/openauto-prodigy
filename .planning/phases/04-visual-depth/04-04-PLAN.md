<task type="auto">
  <name>Task 1: Replace navbar barShadow border with gradient fade</name>
  <files>qml/components/Navbar.qml</files>
  <action>
Replace the existing `Rectangle { id: shadow }` (lines 43-48) with a gradient fade element.

1. Remove the `barShadowColor` property from the navbar (line 17)
2. Remove the old shadow Rectangle (lines 43-48)
3. Add a new gradient Rectangle positioned OUTSIDE the navbar on its content-facing edge:

```qml
Rectangle {
    id: navGradient
    visible: !navbar.aaActive  // Skip during AA -- black navbar blends with phone's dark status bar
    z: -1  // Behind navbar content but above content area

    gradient: Gradient {
        orientation: navbar.isVertical ? Gradient.Horizontal : Gradient.Vertical
        GradientStop {
            position: 0.0
            color: Qt.rgba(navbar.barBg.r, navbar.barBg.g, navbar.barBg.b, 0.6)
        }
        GradientStop { position: 1.0; color: "transparent" }
    }
}
```

4. The gradient must be positioned on the CONTENT-FACING edge of the navbar. Use States with AnchorChanges to handle all 4 edge positions:
   - **bottom navbar** (default): gradient is ABOVE the navbar, `height: Math.round(6 * UiMetrics.scale)`, `width: navbar.width`, anchored with `bottom: navbar.top`
   - **top navbar**: gradient is BELOW, `height: Math.round(6 * UiMetrics.scale)`, `width: navbar.width`, anchored with `top: navbar.bottom`
   - **left navbar**: gradient is to the RIGHT, `width: Math.round(6 * UiMetrics.scale)`, `height: navbar.height`, anchored with `left: navbar.right`
   - **right navbar**: gradient is to the LEFT, `width: Math.round(6 * UiMetrics.scale)`, `height: navbar.height`, anchored with `right: navbar.left`

5. For vertical edges (left/right), the gradient orientation should be `Gradient.Horizontal`. For horizontal edges (top/bottom), use `Gradient.Vertical`. The gradient should fade FROM the navbar edge INTO the content area.

6. For top/left edges: the GradientStop positions may need to be reversed (0.0 = transparent at the far side, 1.0 = navbar color at the near side) depending on the anchor direction. Test that the fade goes from solid at the navbar edge to transparent at the content side.

**Note:** The gradient Rectangle must be a child of the navbar's parent (or the navbar itself positioned outside bounds) so it renders in the content area, not clipped by the navbar. Check if the navbar clips its children -- if so, make the gradient a sibling.
