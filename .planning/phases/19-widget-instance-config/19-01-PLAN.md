<objective>
Add per-instance widget configuration storage to the widget contract: config schema field type definitions, config map on GridPlacement, YAML persistence, model roles, and effective config merging in WidgetInstanceContext.

Purpose: Enables any widget to declare typed config options (enum, bool, int-range) and have per-placement config stored independently, surviving restarts and grid remaps. This is the data layer that the config sheet UI (Plan 02) will write to.

Output: Extended WidgetTypes.hpp with ConfigSchemaField, updated GridPlacement with config map, YAML round-trip for config, WidgetGridModel config roles/methods, WidgetInstanceContext.effectiveConfig property, and comprehensive tests.
</objective>

    `effectiveConfig()` implementation:
    ```cpp
    QVariantMap WidgetInstanceContext::effectiveConfig() const {
        QVariantMap result = defaultConfig_;
        for (auto it = instanceConfig_.constBegin(); it != instanceConfig_.constEnd(); ++it)
            result[it.key()] = it.value();
        return result;
    }
    ```

    `setInstanceConfig()` implementation:
    ```cpp
    void WidgetInstanceContext::setInstanceConfig(const QVariantMap& config) {
        if (instanceConfig_ == config) return;
        instanceConfig_ = config;
        emit effectiveConfigChanged();
    }
    ```

    Updated createContext:
    ```cpp
    QObject* WidgetContextFactory::createContext(const QString& instanceId, QObject* parent) {
        if (!gridModel_) return nullptr;
        const auto& placements = gridModel_->placements();
        for (const auto& p : placements) {
            if (p.instanceId == instanceId) {
                QVariantMap defaultCfg;
                if (auto* reg = gridModel_->registry()) {
                    auto desc = reg->descriptor(p.widgetId);
                    if (desc) defaultCfg = desc->defaultConfig;
                }
                return new WidgetInstanceContext(p, cellSide_, cellSide_, hostContext_,
                                                  defaultCfg, p.config, parent);
            }
        }
        return nullptr;
    }
    ```
