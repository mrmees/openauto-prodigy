  <action>
Replace the entire `settingsList` Component (which contains the ListView + ListModel) with a new Component containing an Item with a centered GridLayout of 6 Tile components.

**Grid structure:**
- `GridLayout` with `columns: 3`, `anchors.centerIn: parent`
- `columnSpacing: UiMetrics.gridGap`, `rowSpacing: UiMetrics.gridGap`
- 6 Tiles, each with `Layout.preferredWidth: UiMetrics.tileW`, `Layout.preferredHeight: UiMetrics.tileH`

**The 6 tiles (order: row 1 left-to-right, row 2 left-to-right):**

1. **Android Auto** — icon: `"\ue531"` (directions_car), subtitle binding:
   ```
   tileSubtitle: { var r = ConfigService.value("video.resolution") || "720p"; var f = ConfigService.value("video.fps") || 60; return r + " " + f + "fps" }
   ```
   onClicked: `openPage("video")` — temporarily uses existing VideoSettings until 02-02 creates AASettings

2. **Display** — icon: `"\ueb97"` (display_settings), subtitle binding:
   ```
   tileSubtitle: { var t = ConfigService.value("display.theme") || "default"; return t.charAt(0).toUpperCase() + t.slice(1) }
   ```
   onClicked: `openPage("display")`

3. **Audio** — icon: `"\ue050"` (volume_up), subtitle binding:
   ```
   tileSubtitle: { var v = typeof AudioService !== "undefined" ? AudioService.masterVolume : 100; return "Vol: " + v + "%" }
   ```
   onClicked: `openPage("audio")`

4. **Connectivity** — icon: `"\ue1d8"` (settings_ethernet), subtitle binding:
   ```
   tileSubtitle: { if (typeof BluetoothManager === "undefined") return ""; var n = BluetoothManager.connectedDeviceName; return n !== "" ? "BT: " + n : "BT: No device" }
   ```
   onClicked: `openPage("connection")` — temporarily uses existing ConnectionSettings until 02-02

5. **Companion** — icon: `"\ue324"` (phonelink), subtitle binding:
   ```
   tileSubtitle: { if (typeof CompanionService === "undefined") return "Disabled"; return CompanionService.connected ? "Connected" : "Not connected" }
   ```
   onClicked: `openPage("companion")`

6. **System** — icon: `"\uf8cd"` (settings_suggest), subtitle binding:
   ```
   tileSubtitle: "v" + (ConfigService.value("identity.sw_version") || "0.0.0")
   ```
   onClicked: `openPage("system")`
