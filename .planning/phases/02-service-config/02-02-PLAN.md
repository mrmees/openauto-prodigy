Add to `YamlConfig.hpp`:
```cpp
// EQ config — dedicated methods (cannot use generic setValueByPath for sequences/maps)
QString eqStreamPreset(const QString& streamName) const;
void setEqStreamPreset(const QString& streamName, const QString& presetName);

struct EqUserPreset {
    QString name;
    std::array<float, 10> gains;
};
QList<EqUserPreset> eqUserPresets() const;
void setEqUserPresets(const QList<EqUserPreset>& presets);
```

Add to `YamlConfig.cpp`:
- `initDefaults()`: add `audio.equalizer.streams.media.preset: "Flat"`, `audio.equalizer.streams.navigation.preset: "Voice"`, `audio.equalizer.streams.phone.preset: "Voice"`, `audio.equalizer.user_presets: []` (empty sequence)
- `eqStreamPreset()`: read from `root_["audio"]["equalizer"]["streams"][streamName]["preset"]`, return as QString, default to "Flat" if missing
- `setEqStreamPreset()`: write to same path
- `eqUserPresets()`: iterate `root_["audio"]["equalizer"]["user_presets"]` sequence, each entry has "name" (string) and "gains" (sequence of 10 floats)
- `setEqUserPresets()`: clear and rebuild the sequence node

**EqualizerService config integration:**

Modify `EqualizerService` constructor to accept `YamlConfig*`:
```cpp
explicit EqualizerService(YamlConfig* config, QObject* parent = nullptr);
```

Add private methods:
