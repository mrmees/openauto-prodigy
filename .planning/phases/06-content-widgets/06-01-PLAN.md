    - Namespace `oap::aa`
    - Q_OBJECT with Q_PROPERTYs: navActive (bool), roadName (QString), maneuverType (int), turnDirection (int), formattedDistance (QString), distanceMeters (int), hasManeuverIcon (bool), iconVersion (int)
    - `connectToHandler(oaa::hu::NavigationChannelHandler* handler)` — connects to handler's navigationStateChanged and navigationTurnEvent signals using Qt::QueuedConnection (handler emits from ASIO thread)

    - `onNavigationStateChanged(bool active)` slot — sets navActive_, clears cached data on false, emits navActiveChanged

    - Inherits QQuickImageProvider(QQuickImageProvider::Image)
    - `requestImage(const QString& id, QSize* size, const QSize& requestedSize)` override — loads currentIcon_ via QImage::loadFromData, scales if requestedSize valid
    - `updateIcon(const QByteArray& pngData)` — stores under QMutex, replaces currentIcon_
    - Thread-safe: QMutex protects currentIcon_ (requestImage called from Qt image thread, updateIcon from main thread)
